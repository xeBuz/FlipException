## Installation

```bash
pip install flipexception
```

## Usage

```python
#!/usr/bin/env python
# -*- coding: utf-8 -*-


from flipexception import FlipException


try:
    raise FlipException("Oops, something happens")
except Exception, e:
    print str(e)
```
